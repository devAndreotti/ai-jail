# v1.4.0

## Arch Linux AUR packaging release

This release adds Arch User Repository packaging support. The project now has co-maintainer access for both live AUR package bases and has published `1.4.0-1` there.

### AUR packages

- Added `packaging/aur/PKGBUILD` for the source package `ai-jail`.
- Added `packaging/aur/PKGBUILD-bin` for the prebuilt package `ai-jail-bin`.
- The binary package installs the Linux x86_64 artifact from GitHub Releases.
- The source package builds locally with Cargo and supports `x86_64` and `aarch64`.
- Both package variants depend on `bubblewrap` and install `ai-jail` to `/usr/bin/`.
- Published both AUR package bases as `1.4.0-1`.

### Documentation

- Added AUR install instructions to the README.
- Added AUR commands to the generated GitHub release body.
- Added `packaging/aur/README.md` with maintainer release steps, checksum helpers, and `.SRCINFO` guidance.
- Added local `.ai-jail` and `.codex` state to `.gitignore`; `.ai-jail` is no longer tracked in the repository.

### Verification

- `bash -n packaging/aur/PKGBUILD`
- `bash -n packaging/aur/PKGBUILD-bin`
- `makepkg --verifysource -p packaging/aur/PKGBUILD`
- `makepkg --verifysource -p packaging/aur/PKGBUILD-bin`
- `makepkg --printsrcinfo -p packaging/aur/PKGBUILD`
- `makepkg --printsrcinfo -p packaging/aur/PKGBUILD-bin`
- `makepkg -Ccf --nodeps -p packaging/aur/PKGBUILD`
- `makepkg -Ccf --nodeps -p packaging/aur/PKGBUILD-bin`
- `cargo fmt --check`
- `cargo clippy -- -D warnings`
- `cargo test`
- `RUSTFLAGS="-D warnings" cargo check --target aarch64-apple-darwin`
